# gatsbyで自分のdev Blogを作った

何か成果物を作りたいなと思いまず始めに自身の学習の記録ができるブログの作成を思いつきました。

記事にしていく内容としては、自分が日常で読んだ本(トラ技、softwareDesignなど)の感想、フィードバックなどを考えていきたいと思います。

とは言っても、最初の記事としてはこのサイトを作成するに当たって役に立ったことなどを紹介していきます。

---
# 技術選定

技術選定の前提としては、  
1. 素早くサイト作成、デプロイが行えること。(簡単)
2. 記事の管理がmarkdownで行えること。　　

の二つを重視して[gatsby.js](https://www.gatsbyjs.com/)を利用しました。

他の選択肢としては、元々pythonをやっていたので[Django](https://www.djangoproject.com/)や[next.js](https://nextjs.org/)などがありましたが、ドキュメントが充実していたのでgatsbyに決めました。
